Esse é um projetinho simples que eu fiz em PHP para organizar dados da Copa do Mundo.
A ideia foi treinar organização de código e estrutura de projeto, então não é nada super complexo, é mais pra estudo e apresentação mesmo.
Com ele dá pra cadastrar e visualizar:
Seleções
Usuários
Grupos
Jogos
Resultados
Classificação
Tecnologias que usei
PHP (com um MVC bem simples)
MySQL
CSS
XAMPP
Como rodar o projeto
Coloca a pasta do projeto dentro do htdocs
(ex: C:/xampp/htdocs/06.02.2026)
Abre o XAMPP e inicia o Apache e o MySQL
Cria um banco de dados no MySQL com o nome cop_word
Confere se os dados de conexão estão certos em app/config/config.php
Abre no navegador:
http://localhost/06.02.2026/public/index.php
Banco de dados
